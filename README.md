### 一、NEED
>需要安装的库

mac系统下
```
brew install pkg-config
brew install imagemagick
```

golang的库
```
go get gopkg.in/gographics/imagick.v3/imagick
pkg-config --cflags --libs MagickWand

go get github.com/BurntSushi/toml
```
